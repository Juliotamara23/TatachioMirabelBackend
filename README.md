<h1 align="center">Backend para Cabildo Tatachio Mirabel</h1>

## Descripción del Proyecto

Este proyecto es el backend para la gestión de miembros del cabildo Tatachio Mirabel. Su objetivo principal es actuar como un **SaaS (Software as a Service)** para automatizar la gestión de datos censales, realizar análisis de consistencia de la información y generar reportes bajo los estándares exigidos por el **Ministerio del Interior de Colombia**.

### Tecnologías Principales
- **Lenguaje:** TypeScript
- **Entorno:** Node.js
- **Framework:** Express
- **ORM:** Prisma
- **Validación:** Zod
- **IA:** Vercel AI SDK (Integración con modelos de Google y Ollama)

### Características Clave
- **Gestión de Datos:** CRUD completo para miembros, familias y grupos del cabildo.
- **Validación Estricta:** Cumplimiento riguroso de formatos censales gubernamentales (fechas, códigos, documentos).
- **Inteligencia Artificial:** Análisis automático para detectar inconsistencias y duplicados.
- **Reportes Automatizados:** Generación de formatos Excel oficiales para el Ministerio del Interior.
